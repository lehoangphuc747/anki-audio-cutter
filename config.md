# Audio Card Cutter — Configuration

- **audio_field**: Name of the note field that receives the `[sound:...]` tag. Default: `Audio`.
- **default_deck**: Default deck when creating notes. Leave blank to use the current deck.
- **default_notetype**: Default Note Type. Leave blank to use the current note type.
- **output_format**: Output format (`mp3`, `ogg`, `wav`, `m4a`). Default: `mp3`.
- **output_bitrate**: Audio bitrate (e.g. `128k`, `192k`).
- **ffmpeg_path**: Absolute path to `ffmpeg` if not in PATH. Leave blank for auto-detection.
- **language**: UI language (`en`, `vi`, etc.). Leave blank to auto-detect from Anki.
- **last_audio_dir**: *(auto-saved)* Last directory used for file picker.
- **last_tags**: *(auto-saved)* Last tags entered.
